# Breakout Game

HTML5 implementacija arkadne igre Breakout.

## Struktura

- `index.html` - glavna HTML datoteka
- `styles.css` - CSS stilovi
- `game.js` - JavaScript logika igre

## Funkcionalnosti

- Canvas 800×600px, crna pozadina, bijeli rub 5px
- Početni ekran sa "BREAKOUT" i "Press SPACE to begin"
- 50 cigli u 5 redova i 10 stupaca
- Loptica i palica sa 3D efektom
- Detekcija kolizija
- Rezultati sa localStorage
- GAME OVER i YOU WIN ekrani
- Upravljanje strelicama ili A/D tipkama

## Pokretanje

1. Otvori `index.html` u web pregledniku
2. Pritisni SPACE za početak igre
3. Koristi strelice ili A/D za upravljanje palicom
